# hello-world-js
Meu Primeiro Código em JavaScript

Este repositório contém meu primeiro projeto em JavaScript: um pequeno programa que solicita o nome do usuário e exibe uma saudação personalizada.

## Como executar
1. Baixe ou clone o repositório.
2. Abra o arquivo `index.html` no navegador.
3. Interaja com o prompt e com o botão para testar o código.

## Objetivo
Aprender conceitos básicos de programação, versionamento e uso do GitHub.
